Iterate over a map of tuples (particle_line_contact_candidates) to see if the particle-line pairs are in contact or not. If they are in contact, the normal overlap, normal vector of contact and contact normal relative velocity are stored in a map which is the output of this function.

Parameters
particle_line_contact_candidatesThe output of particle-line broad search which shows contact pair candidates.
neighborhood_thresholdA value which defines the neighbor particles.
particle_line_pairs_in_contactA map which contains all the required information (normal overlap, normal vector and contact normal relative velocity) for calculation of particle-line contact force.
