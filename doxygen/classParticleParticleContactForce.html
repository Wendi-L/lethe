Detailed Description

template<int dim, Parameters::Lagrangian::ParticleParticleContactForceModel contact_model, Parameters::Lagrangian::RollingResistanceMethod rolling_friction_model>
class ParticleParticleContactForce< dim, contact_model, rolling_friction_model >

Execute calculation of particle-particle contact forces including linear, non-linear contact models, and non-linear with cohesive forces.

Instead of using a inheritance hierarchy to distinguish between the contact model, the class is templated with the type of force model and rolling friction model. Consequently, the code for each combination of force model is generated at compile time.

Template Parameters
dimThe dimension of the problem.
force_modelThe particle-particle contact force model.
rolling_friction_modelThe rolling resistance model.
