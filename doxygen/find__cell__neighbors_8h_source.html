15 /**
16  * @brief Finds the neighbor list (without repetition) of all the active
17  * cells in the triangulation. It gets the vertices of the cells to get lists
18  * of the neighbor for each cells. There is some check to prevent repetition
19  * of a cell in a list (up to 8 vertices can have the same cell in common in
20  * 3D).
21  * 2 types of container are used for cell neighbors : local-local cells
22  * and local-ghost cells.
23  *
24  * @param triangulation Triangulation to access the information of the cells
25  * @param cells_local_neighbor_list A vector (with size of the local cell
26  * number) of vectors (local adjacent cells of each local cell). First element
27  * of each set shows the main cell itself
28  * @param cells_ghost_neighbor_list A vector (with size of the local cell
29  * number) of vectors (ghost adjacent cells of each local cell). First element
30  * of each set shows the main cell itself
31  */

41 /**
42  * @brief Finds the periodic neighbor list (without repetition) of all the
43  * active cells in the triangulation. It gets the coinciding vertices of the
44  * cells at the periodic boundary 0 to get lists of the periodic neighbor
45  * cells on the periodic boundary 1 for each cells. There is some check to
46  * prevent repetition of a cell in a list (up to 8 vertices can have the
47  * same cell in common in 3D).
48  * 3 types of container are used for periodic mapping of cell neighbors :
49  * local-local cells, local-ghost cells and ghost-local cells. The last
50  * container is necessary since the mapping are only from periodic boundary 0
51  * to periodic boundary 1 and the ghost-local particle pairs need distinction
52  * for proper handling of search of particle pairs and contact forces.
53  *
54  * @param triangulation Triangulation to access the information of the cells
55  * @param periodic_boundaries_cells_information A container of information
56  * related to the pairs of cell at periodic boundaries, used to get the cells
57  * on periodic boundary 0
58  * @param cells_local_periodic_neighbor_list A vector (with size of the local
59  * cell number) of vectors (local adjacent cells of each local cell). First
60  * element of each set shows the main cell itself
61  * @param cells_ghost_periodic_neighbor_list A vector (with size of the local
62  * cell number) of vectors (ghost adjacent cells of each local cell). First
63  * element of each set shows the main cell itself
64  * @param cells_ghost_local_periodic_neighbor_list A vector (with size of the
65  * ghost cell number) of vectors (local adjacent cells of each ghost cell).
66  * First element of each set shows the main ghost cell itself
67  */

99 /**
100  * @brief Generate a periodic neighbor cells list of the cell. With the
101  * coinciding_vertex_groups and the vertex_to_coinciding_vertex_group, we can
102  * get the coinciding vertices on a periodic boundary and with v_to_c we can
103  * get the list of the periodic neighbor cells to the main cell.
104  *
105  * Here is an example to understand how the search works :
106  * Cell 8 have vertices 0, 1, 2, 3 on the periodic boundary. First,
107  * it checks in the coinciding_vertex_groups if the vertex 0 is a key in the
108  * map, and if it is, it gets a label, let's say label is 10. In the
109  * vertex_to_coinciding_vertex_group, it can finds the the coinciding vertices
110  * with the label. Using the label 10, it gets the vertices 0 and 34, which
111  * means that vertices 0 and 34 are periodic. We do not want the cells
112  * attached to the vertex 0 since they are already found with the regular
113  * find_cell_neighbor function, so it skips vertex 0, but gets the cells 21,
114  * 22, 23, 24 attached to the vertex 34. The same checks are done for vertices
115  * 1, 2 and 3 and all periodic cells are return as a vector.
116  *
117  * @param cell The cell that needs the periodic neighbor list
118  * @param coinciding_vertex_groups A map of coinciding vertices labeled by an
119  * arbitrary element from them
120  * @param vertex_to_coinciding_vertex_group Map of a vertex to the label of a
121  * group of coinciding vertices
122  * @param v_to_c A vector of set with adjacent cells of all the vertices
123  * @param periodic_neighbor_list A vector which is the list of periodic cell
124  * neighbors
125  */
126 template <int dim>
127 void
128 get_periodic_neighbor_list(
129  const typename Triangulation<dim>::active_cell_iterator &cell,
130  const std::map<unsigned int, std::vector<unsigned int>>
131  &coinciding_vertex_groups,
132  const std::map<unsigned int, unsigned int> &vertex_to_coinciding_vertex_group,
133  const std::vector<std::set<typename Triangulation<dim>::active_cell_iterator>>
134  &v_to_c,
135  typename DEM::dem_data_structures<dim>::cell_vector &periodic_neighbor_list);
