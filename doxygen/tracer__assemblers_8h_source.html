28  * @brief Class that assembles the core of the Tracer equation.
29  * This class assembles the weak form of:
30  * \f$\mathbf{u} \cdot \nabla T - D \nabla^2 =0 \f$ with an SUPG
31  * stabilization
32  *
33  * @tparam dim An integer that denotes the number of spatial dimensions
34  *
35  * @ingroup assemblers
36  */

39 {
40 public:
