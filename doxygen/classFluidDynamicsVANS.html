◆ get_periodic_offset_distance()

template<int dim>
Tensor<1, dim> FluidDynamicsVANS< dim >::get_periodic_offset_distance ( unsigned int  boundary_id) const
inlineprivate

This function calculates and returns the periodic offset distance of the domain which is needed for the periodic boundary conditions using the QCM or SPM for void fraction with the GLS VANS/CFD-DEM solver. The distance is based on one of the periodic boundaries and all particle location shifted by this distance is according to this periodic boundary.

Parameters
boundary_idThe id of one of the periodic boundaries
Returns
The periodic offset distance
